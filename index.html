Background

However, with the recent trend of the world moving towards open source JavaScript frameworks (and Microsoft promoting the same for SharePoint with all the add-ins), I decided not to get left behind. Instead, I am adapting by learning JavaScript/HTML5/CSS3 open source frameworks. My last few projects involved extensive web front end development on sites hosted on SharePoint. I used JavaScript and CSS frameworks/libraries like jQuery (heavy usage of Ajax), Bootstrap, DropzoneJS, PaceJS, Numeral, X-Editable etc. to build responsive web applications. Even handled the server side development by building RESTful WCF services.

Essentially "Full Stack" development but with SharePoint playing the "Database" role and .NET WCF being the middle/server layer. Along with development, I even do "web design". This is due to my natural creative/artistic skills with a keen eye for detail. That said, I am a developer first and foremost with design skills just an added bonus.
